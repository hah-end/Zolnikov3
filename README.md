# Описание проетка

Weather Diary — это графическое приложение на Python, позволяющее вести дневник погоды. Пользователь может:
- Добавлять записи с указанием даты, температуры, описания погоды и наличия осадков
- Просматривать все записи в виде таблицы
- Фильтровать данные по дате и температуре (например, показать дни теплее +10 °C)
- Сохранять данные в файл `weather_data.json`
- Загружать ранее сохранённые записи при запуске
- Проверять корректность ввода: дата в формате ГГГГ-ММ-ДД, температура — число, описание — не пустое

Приложение построено с использованием встроенных библиотек Python: `tkinter` для графического интерфейса и `json` для хранения данных.

## Требования

* Python 3.6+
* Библиотеки: `tkinter` (встроен в Python), `json` (встроен в Python)
